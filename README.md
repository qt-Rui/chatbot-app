# Gemini Chatbot (Next.js)

A simple AI chatbot built with:

- Next.js
- TypeScript
- Google Gemini API
- React
- TailwindCSS

## Setup

1. Clone the repo
2. Install dependencies:

npm install

3. Create `.env.local`

GEMINI_API_KEY=your_key_here

4. Run dev server:

npm run dev

## Notes
- API keys are never exposed to the browser.
- Basic rate limiting is implemented.
